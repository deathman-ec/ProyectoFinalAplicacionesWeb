# ProyectoFinalAplicacionesWeb
Sistema de Gestión de Inventario

Proyecto 03 - Programación Web | Spring Boot 3 + Java 21 + MariaDB + Thymeleaf

## Descripción
Aplicación web para controlar stock de productos en múltiples sucursales. 
Registra entradas, salidas y ajustes de inventario con trazabilidad completa.

## Stack Tecnológico
- Java 21
- Spring Boot 3.2.5
- Spring Data JPA / Hibernate
- PostgreSQL 15+
- Thymeleaf 3
- Maven 3.9+
- JUnit 5

## Requisitos Previos
- JDK 21 instalado
- Maven 3.9+
- MariaDB 
- VS Code con extensiones: Extension Pack for Java, Spring Boot Extension Pack
